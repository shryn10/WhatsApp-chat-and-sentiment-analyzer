IN VS CODE:-

1)Create a Virtual Environment (Recommended)

  Inside your project folder, open terminal (Ctrl+`) and run:
    python -m venv venv

  Then activate it:

  On Windows:
    .\venv\Scripts\activate

  On Mac/Linux:
    source venv/bin/activate

  Then exit the environment by typing: 
    exit()

2)Install the required things, run this code in the terminal:
    pip install pandas numpy scikit-learn imbalanced-learn matplotlib
